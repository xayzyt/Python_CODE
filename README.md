# Python Practice

Python 辅助工具与语法练习仓库。它和我的嵌入式方向互补：C 负责贴近硬件，Python 用来快速验证想法、处理文件、整理数据和写一些效率脚本。

## 仓库定位

这个仓库主要记录 Python 基础练习，不是完整产品项目。它体现的是我用脚本解决小问题的能力：

- 快速写验证代码。
- 处理文件、文本和简单数据。
- 练习包、模块、导入关系和脚本组织。
- 为后续嵌入式项目准备辅助工具能力。

## 内容导览

| 目录 | 内容 |
| --- | --- |
| `pythonProjectpy1` ~ `pythonProjectpy4` | Python 基础语法练习 |
| `pythonProjectpy5` | 包、模块、登录/注册等简单组织练习 |
| `pythonProjectpy9` | 自定义 package / module 练习 |
| `pythonProjectpy10` | 脚本与 HTML 渲染相关练习 |
| `test.py` | 零散验证脚本入口 |

## 和嵌入式方向的关系

我不会把 Python 当作主力嵌入式开发语言，但它很适合做这些事情：

- 批量生成或转换配置文件。
- 解析串口日志、HTTP 返回数据或传感器数据。
- 快速验证算法和通信协议格式。
- 为调试、测试和数据整理写小工具。
